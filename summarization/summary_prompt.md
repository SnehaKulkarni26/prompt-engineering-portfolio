# Text Summarization Prompt

## Goal
Summarize long text into short and simple points.

## Prompt (Version 1)
"Summarize the following text into 3 simple bullet points using easy language."

## Output (Version 1)
Read the text below and summarize it into exactly 3 bullet points.
Use simple words, short sentences, and easy language so a beginner can understand.

## Observation (Version 1)
- Bullet points made the summary easier to read.
- Simple language reduced complexity.

---

## Prompt (Version 2 – Small Change)
"Summarize the following text into 3 clear bullet points for a beginner audience."

## Output (Version 2)
“Summarize the following text into exactly 3 clear bullet points using simple, beginner-friendly language.”

## Observation (Version 2)
- Audience instruction improved clarity.
- Output became more structured.

---

## What I Learned
- Changing audience affects summarization style.
- Clear constraints improve AI responses.
